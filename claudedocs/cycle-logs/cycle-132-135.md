# Cycles 132-135: f64 Math Intrinsics + Codegen 품질 개선

Date: 2026-03-28

## Scope & Implementation

### f64 수학 인트린식 추가 (v0.97.1)

6개 LLVM 인트린식을 BMB 빌트인으로 노출:

| 함수 | LLVM 인트린식 | 용도 |
|------|-------------|------|
| `sin(x: f64) -> f64` | `@llvm.sin.f64` | 삼각함수 |
| `cos(x: f64) -> f64` | `@llvm.cos.f64` | 삼각함수 |
| `floor(x: f64) -> f64` | `@llvm.floor.f64` | 내림 |
| `ceil(x: f64) -> f64` | `@llvm.ceil.f64` | 올림 |
| `fabs(x: f64) -> f64` | `@llvm.fabs.f64` | 절대값 |
| `pow_f64(x: f64, y: f64) -> f64` | `@llvm.pow.f64` | 거듭제곱 |

### 변경 파일

1. **bmb/src/types/mod.rs**: 타입 시스템에 6개 함수 시그니처 등록
2. **bmb/src/codegen/llvm_text.rs**: LLVM IR 코드젠 핸들러 추가
3. **bmb/src/interp/eval.rs**: 인터프리터 빌트인 함수 6개 추가
4. **bmb/src/codegen/llvm_text.rs**: calloc 전용 코드젠 핸들러 (Cycle 131)
