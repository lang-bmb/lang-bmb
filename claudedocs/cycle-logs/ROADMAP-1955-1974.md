# Roadmap: Cycles 1955-1974 — gotgan-packages → Binding Libraries

> **Scope**: gotgan-packages의 기존 102 패키지를 활용하여 바인딩 라이브러리 빌드
> **Strategy**: 기존 코드에 @export 추가 → 공유 라이브러리 → Python/JS 바인딩

## Phase A: gotgan-packages @export 적용 (Cycles 1955-1958)

| Cycle | Target |
|-------|--------|
| 1955 | bmb-dp: lcs, edit_distance, knapsack, coin_change에 @export 추가 + 빌드 |
| 1956 | bmb-sort: quick_sort, merge_sort, heap_sort에 @export 추가 + 빌드 |
| 1957 | bmb-graph: bfs, dijkstra, scc에 @export 추가 + 빌드 |
| 1958 | bmb-matrix: multiply, transpose, determinant에 @export 추가 + 빌드 |

## Phase B: 통합 bmb-algo 라이브러리 (Cycles 1959-1962)

| Cycle | Target |
|-------|--------|
| 1959 | gotgan-packages에서 최적 함수 선별 → 단일 lib.bmb로 통합 |
| 1960 | 통합 라이브러리 빌드 + 벤치마크 vs C |
| 1961 | Python 바인딩 확장 (ctypes + 벤치마크 스크립트) |
| 1962 | README + 문서 + "Powered by BMB" |

|-------|--------|
| 1963 | 전체 테스트 + Stage 1 부트스트랩 |
| 1964-1974 | 추가 패키지, 정리, 최종 검증 |
