### Bug Fixed: heap_sort recursive stack overflow
**Symptom**: heap_sort crashes with access violation for n > ~800
**Root cause**: hs_heapify was recursive, causing stack overflow for large heaps
**Fix**: Converted hs_heapify from recursive to iterative (while loop)
**Limitation**: BMB runtime memory still limits practical input size. Documented in test_stress.py.

### Discovered: b64_chr control character limitation
- hex_decode and rot13 fail for control characters (0x00-0x1F) because b64_chr lookup table only covers printable ASCII (32-126)
- Documented in stress tests, test cases limited to printable ASCII
