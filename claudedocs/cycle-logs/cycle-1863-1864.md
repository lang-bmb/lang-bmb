## Scope & Implementation

### 1. Bootstrap `noundef` Attribute Sync (Cycle 1863)
Added `noundef` to all function parameters and return types in bootstrap compiler:
- `gen_param()`: `llvm_type(param_type) + " noundef %" + param_name` for all non-String params
- `gen_param()`: String params get `i8* noundef nonnull noalias nocapture readonly dereferenceable(24) align 8 %name`
- `gen_fn_header()`: Return types get `noundef` prefix (+ `nonnull` for String returns, nothing for void)
- **Note**: String-specific attributes (nonnull, noalias, etc.) don't fire because MIR represents String as `i64` at codegen level. This requires MIR type propagation to fix (higher effort).
- **Result**: 1,450 `noundef` annotations added to IR; IR size 107,729 тЖТ 107,567 lines
