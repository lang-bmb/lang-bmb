### Analysis
- ~540 runtime symbols are shared across all libraries (arc, channel, barrier, array ops, string ops, etc.)
- Only 10-43 symbols per library are the actual @export functions
- DLL sizes are 272-304 KB — reasonable for LLVM-compiled binaries with static runtime

### Symbol filtering options (not implemented — requires compiler changes)
1. **Linker .def file**: Export only @export symbols
2. **Visibility attributes**: `__attribute__((visibility("hidden")))` for runtime
3. **LTO + internalize**: LLVM's `-internalize-public-api-list` pass

These are compiler-level changes, not binding-level changes. Documented for future work.
