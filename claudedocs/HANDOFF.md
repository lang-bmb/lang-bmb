## 새로 추가된 언어 기능 요약

### M12: Effect Row 문법 (`fn foo(): <effects> -> T`)
```bmb
fn double(x: i64): <pure> -> i64 = x * 2;   // fn-pure 노드 (기존 @pure와 동등)
fn read_line(): <IO> -> String = ...;         // (eff IO) 자식, "bmb-effect"="IO" LLVM
fn any(n: i64): <*> -> i64 = n;              // 제한 없음 (기존과 동일)
fn multi(x: i64): <IO, Net> -> i64 = ...;   // "bmb-effect"="IO Net" LLVM
```

### M13: Intent 어노테이션
```bmb
fn add(x: i64, y: i64) -> i64
    intent: "Adds two non-negative numbers"
    pre x >= 0 and y >= 0
    post it >= 0
= x + y;
```

`bmb lint`: `[intent_no_contract]` 경고 - intent 있는데 pre/post 없으면 경고.
