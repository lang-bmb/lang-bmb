# 183. Array Partition I

Given an integer array of 2n integers, group these integers into n pairs (a1, b1), (a2, b2), ..., (an, bn) such that the sum of min(ai, bi) for all i is maximized. Return the maximized sum. (external, LeetCode #561)

Input: first line n, then 2n integers (one per line)
Output: the maximized sum

Example:
Input:
2
1
4
3
2
Output: 4

Explanation: Pairs (1,2) and (3,4). Sum of mins = 1+3 = 4.

Input:
3
6
2
6
5
1
2
Output: 9
