# 171. Nim Game

You are playing the following Nim Game: there is a heap of stones, and you take turns removing 1 to 3 stones. The player who removes the last stone wins. Given n stones, determine if you can win assuming both players play optimally. (external, LeetCode #292)

Input: n (number of stones)
Output: 1 if you can win (first player), 0 otherwise

Example:
Input: 4
Output: 0
(4 stones: whatever you take (1,2,3), opponent can always win)

Input: 1
Output: 1

Input: 7
Output: 1
