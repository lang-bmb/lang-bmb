# 266. Check If String Is a Prefix of Array

Given a string s and an array of strings words, return 1 if s is a prefix string of words (s is equal to the concatenation of the first k strings in words for some positive k), or 0 otherwise. (external, LeetCode #1961)

Input: first line is s, second line is n (number of words), then n words (one per line)
Output: 1 if s is a prefix string, else 0

Example:
Input:
iloveleetcode
4
i
love
leetcode
im
Output:
1

Example:
Input:
iloveleetcode
4
apples
i
love
leetcode
Output:
0
