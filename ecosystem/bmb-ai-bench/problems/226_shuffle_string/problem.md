# 226. Shuffle String

Given a string s and an integer array indices of the same length, the string s will be shuffled such that the character at the ith position moves to indices[i] in the shuffled string. Return the shuffled string. (external, LeetCode #1528)

Input: first line the string s, then n integers (one per line, where n = length of s)
Output: the shuffled string

Example:
Input:
codeleet
4
5
6
7
0
2
1
3
Output:
leetcode

Input:
abc
0
1
2
Output:
abc
