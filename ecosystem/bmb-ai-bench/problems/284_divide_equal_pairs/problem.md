# 284. Divide Array Into Equal Pairs

Given an integer array nums of even length n, divide it into n/2 pairs. Return 1 if the array can be divided into n/2 pairs where each pair consists of equal elements, else 0. (external, LeetCode #2206)

Input: first line is n, then n integers (one per line)
Output: 1 if possible, else 0

Example:
Input:
4
3
2
3
2
Output:
1

Example:
Input:
4
1
2
3
4
Output:
0
