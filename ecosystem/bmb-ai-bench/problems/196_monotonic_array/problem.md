# 196. Monotonic Array

An array is monotonic if it is either entirely non-increasing or non-decreasing. Given an array, return 1 if monotonic, 0 otherwise. (external, LeetCode #896)

Input: first line n, then n integers (one per line)
Output: 1 if monotonic, 0 otherwise

Example:
Input:
4
1
2
2
3
Output: 1

Input:
3
6
5
4
Output: 1

Input:
3
1
3
2
Output: 0
