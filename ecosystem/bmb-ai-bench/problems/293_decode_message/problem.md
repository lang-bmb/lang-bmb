# 293. Minimum Amount of Time to Fill Cups

You have a water dispenser that fills one cup per second. In each second, you can fill one cup of each of two different types. Given amounts of three types (cold, warm, hot), return the minimum seconds to fill all cups. (external, LeetCode #2335)

Input: three lines: cold, warm, hot amounts
Output: minimum seconds

Example:
Input:
1
4
2
Output:
4

Example:
Input:
5
4
4
Output:
7

Example:
Input:
5
0
0
Output:
5
