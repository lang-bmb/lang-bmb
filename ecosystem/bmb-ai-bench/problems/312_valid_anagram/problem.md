# 312. Valid Anagram

Given two strings s and t, return 1 if t is an anagram of s, otherwise 0. An anagram uses exactly the same letters with the same frequencies. (external, LeetCode #242)

Input: line 1 is s, line 2 is t (lowercase letters)
Output: 1 if anagram, else 0

Example:
Input:
anagram
nagaram
Output:
1
