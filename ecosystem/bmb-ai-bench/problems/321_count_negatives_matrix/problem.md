# 321. Count Negative Numbers in a Matrix

Given an m x n matrix, return the number of negative numbers in it. (external, LeetCode #1351)

Input: line 1 is m, line 2 is n, then m*n integers in row-major order (one per line)
Output: the count of negative numbers

Example:
Input:
2
2
3
2
1
-1
Output:
1
