# 289. Check if Number Has Equal Digit Count and Digit Value

A digit string is valid if for every digit d that appears, the string has exactly d occurrences of digit d... but more precisely: s[i] equals the count of digit i in the string. Given a string num, return 1 if valid, else 0. (external, LeetCode #2283)

Input: one line, digit string num
Output: 1 if valid, else 0

Example:
Input:
1210
Output:
1

Example:
Input:
030
Output:
0
