# 213. How Many Numbers Are Smaller Than the Current Number

Given an array of integers nums, for each element, count how many numbers in the array are smaller than it. (external, LeetCode #1365)

Input: first line n, then n integers (one per line)
Output: for each element, its smaller-count (one per line, in order)

Example:
Input:
4
8
1
2
2
Output:
3
0
1
1

Input:
3
6
5
4
Output:
2
1
0
