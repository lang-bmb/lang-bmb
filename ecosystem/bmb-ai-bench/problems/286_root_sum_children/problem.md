# 286. Root Equals Sum of Children

Given a binary tree with exactly three nodes: root, left child, and right child. Return 1 if root.val == left.val + right.val, else 0. (external, LeetCode #2236)

Input: three lines: root, left, right
Output: 1 if root == left + right, else 0

Example:
Input:
10
4
6
Output:
1

Example:
Input:
5
3
1
Output:
0
