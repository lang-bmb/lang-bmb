# 294. Maximum Number of Pairs in Array

Given a 0-indexed integer array nums, repeatedly remove pairs of equal elements. Return the number of pairs removed and the number of leftover elements. (external, LeetCode #2341)

Input: first line is n, then n integers (one per line)
Output: two numbers: pairs and leftovers, space-separated

Example:
Input:
6
1
3
2
1
3
2
Output:
3 0

Example:
Input:
5
1
1
2
1
2
Output:
2 1
