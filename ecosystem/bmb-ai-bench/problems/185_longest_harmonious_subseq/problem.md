# 185. Longest Harmonious Subsequence

A harmonious array is one where the difference between its maximum and minimum values is exactly 1. Find the length of the longest harmonious subsequence among all subsequences. (external, LeetCode #594)

Input: first line n, then n integers (one per line)
Output: length of longest harmonious subsequence

Example:
Input:
8
1
3
2
2
5
2
3
7
Output: 5

Explanation: longest harmonious subsequence is [3,2,2,2,3], length 5.

Input:
3
1
2
3
Output: 2
