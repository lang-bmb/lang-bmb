# 217. Average Salary Excluding the Minimum and Maximum Salary

Given an array of unique integers representing salaries, return the average salary of all employees excluding the minimum and maximum salary. (external, LeetCode #1491)

Output the result rounded to the nearest integer (floor).

Input: first line n, then n integers (one per line)
Output: floor of average salary excluding min and max

Example:
Input:
4
4000
3000
1000
2000
Output:
2500

Input:
5
1000
2000
3000
4000
5000
Output:
3000
