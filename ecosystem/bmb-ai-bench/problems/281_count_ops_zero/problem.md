# 281. Count Operations to Obtain Zero

You are given two non-negative integers num1 and num2. In one operation, if num1 >= num2, subtract num2 from num1, else subtract num1 from num2. Return the number of operations required to make either num1 or num2 equal to 0. (external, LeetCode #2169)

Input: two lines, num1 then num2
Output: number of operations

Example:
Input:
2 3
Output:
3

Example:
Input:
10 10
Output:
1
