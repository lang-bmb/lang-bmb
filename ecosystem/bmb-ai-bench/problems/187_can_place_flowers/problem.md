# 187. Can Place Flowers

You have a long flowerbed in which some of the plots are planted and some are not. Flowers cannot be planted in adjacent plots. Given an integer array flowerbed and an integer n, return 1 if n new flowers can be planted without violating the no-adjacent-flowers rule, 0 otherwise. (external, LeetCode #605)

Input: first line "k n" where k=flowerbed size, n=flowers to plant, then k integers (0 or 1, one per line)
Output: 1 if possible, 0 otherwise

Example:
Input:
5 1
1
0
0
0
1
Output: 1

Input:
5 2
1
0
0
0
1
Output: 0
