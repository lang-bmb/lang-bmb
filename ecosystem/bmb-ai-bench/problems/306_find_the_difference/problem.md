# 306. Find the Difference

You are given two strings s and t. String t is generated by shuffling string s and then adding one more lowercase letter at a random position. Return the letter that was added to t. (external, LeetCode #389)

Input: line 1 is s (may be empty), line 2 is t
Output: the single extra character

Example:
Input:
abcd
abcde
Output:
e
