# 218. Can Make Arithmetic Progression From Sequence

Given an array of numbers arr, return 1 if arr can be rearranged to form an arithmetic progression, 0 otherwise. (external, LeetCode #1502)

An arithmetic progression is a sequence where the difference between any two consecutive elements is the same.

Input: first line n, then n integers (one per line)
Output: 1 if possible, 0 otherwise

Example:
Input:
3
3
5
1
Output:
1

Input:
3
1
2
4
Output:
0
