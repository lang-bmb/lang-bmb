# 299. Find Subarrays With Equal Sum

Given a 0-indexed integer array nums, return 1 if there exist two consecutive subarrays of length 2 that have the same sum, or 0 otherwise. (external, LeetCode #2395)

Input: first line is n, then n integers (one per line)
Output: 1 if exists, else 0

Example:
Input:
5
3
2
1
6
5
Output:
0

Example:
Input:
5
0
0
0
0
0
Output:
1
