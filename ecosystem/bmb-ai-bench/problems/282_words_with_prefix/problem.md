# 282. Counting Words With a Given Prefix

Given an array of strings words and a string pref, return the number of strings in words that contain pref as a prefix. (external, LeetCode #2185)

Input: first line is n (number of words), second line is pref, then n words (one per line)
Output: count of words with given prefix

Example:
Input:
6
pay
pay
attention
practice
attend
a
a
Output:
1

Example:
Input:
3
a
aa
aaa
aaaa
Output:
3
