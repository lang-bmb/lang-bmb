# 288. Percentage of Letter in String

Given a string s and a character letter, return the percentage of characters in s that equal letter rounded down to the nearest whole percent. (external, LeetCode #2278)

Input: first line is s, second line is the letter
Output: percentage (floor)

Example:
Input:
foobar
o
Output:
33

Example:
Input:
jjjj
k
Output:
0
