# 333. Count All-Positive Rows

Given an m x n matrix, return the number of rows in which every element is strictly positive. (external-style, nested-vector exercise)

Input: line 1 is m, line 2 is n, then m*n integers in row-major order (one per line)
Output: the count of all-positive rows

Example:
Input:
2
2
1
2
3
-1
Output:
1
