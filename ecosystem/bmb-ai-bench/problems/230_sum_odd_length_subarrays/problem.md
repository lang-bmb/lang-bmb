# 230. Sum of All Odd Length Subarrays

Given an array of positive integers arr, calculate the sum of all possible odd-length subarrays. (external, LeetCode #1588)

A subarray is a contiguous subsequence of the array. An odd-length subarray has length 1, 3, 5, ...

Input: first line n, then n integers
Output: the sum

Example:
Input:
5
1
4
2
5
3
Output:
58

Explanation: Odd subarrays: [1]=1, [4]=4, [2]=2, [5]=5, [3]=3, [1,4,2]=7, [4,2,5]=11, [2,5,3]=10, [1,4,2,5,3]=15. Sum=58.
