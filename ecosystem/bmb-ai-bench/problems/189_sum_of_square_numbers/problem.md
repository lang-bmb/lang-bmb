# 189. Sum of Square Numbers

Given a non-negative integer c, decide whether there are two integers a and b such that a*a + b*b = c. Output 1 if yes, 0 otherwise. (external, LeetCode #633)

Input: one integer c
Output: 1 if c = a*a + b*b for some non-negative integers a, b, else 0

Example:
Input: 5
Output: 1
(1*1 + 2*2 = 5)

Input: 3
Output: 0
