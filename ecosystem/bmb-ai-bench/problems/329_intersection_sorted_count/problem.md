# 329. Intersection of Two Sorted Arrays

Given two sorted integer arrays, return the number of distinct values that appear in both. (external, LeetCode #349 variant)

Input: line 1 is n, then n integers (sorted, one per line); next line is m, then m integers (sorted)
Output: the count of distinct common values

Example:
Input:
4
1
2
2
3
3
2
2
2
Output:
1
