# 283. Cells in a Range on an Excel Sheet

Given a string s representing a range of cells in an Excel sheet (format "ColRow:ColRow"), return all cells in that range sorted by column then row. Output space-separated. (external, LeetCode #2194)

Input: one line, range string like "K1:L2"
Output: cells space-separated

Example:
Input:
K1:L2
Output:
K1 K2 L1 L2

Example:
Input:
A1:F1
Output:
A1 B1 C1 D1 E1 F1
