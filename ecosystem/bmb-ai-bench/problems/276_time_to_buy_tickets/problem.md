# 276. Time Needed to Buy Tickets

There are n people in a queue. The ith person wants to buy tickets[i] tickets. People buy one ticket per turn and go to the back of the queue. Given k (0-indexed), return the time taken to buy all of person k's tickets. (external, LeetCode #2073)

Input: first line is n, second line is k, then n ticket counts (one per line)
Output: time for person k to finish buying

Example:
Input:
5
2
2
3
2
2
1
Output:
8

Example:
Input:
3
2
5
1
1
Output:
3
