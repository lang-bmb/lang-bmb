# 271. Final Value of Variable After Performing Operations

There is a programming language with only four operations on variable x: ++X, X++, --X, X--. Given an array of operations, return the final value of x. (external, LeetCode #2011)

Input: first line is n, then n operations (one per line)
Output: final value of x

Example:
Input:
3
--X
X++
X++
Output:
1

Example:
Input:
5
X++
++X
--X
X--
--X
Output:
-1
