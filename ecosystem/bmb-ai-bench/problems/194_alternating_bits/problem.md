# 194. Binary Number with Alternating Bits

Given a positive integer, check whether it has alternating bits: namely, if two adjacent bits will always have different values. Output 1 if yes, 0 otherwise. (external, LeetCode #693)

Input: one positive integer
Output: 1 if alternating bits, 0 otherwise

Example:
Input: 5
Output: 1
(5 = 101 in binary)

Input: 7
Output: 0
(7 = 111 in binary)

Input: 11
Output: 0
(11 = 1011, adjacent bits 1,0,1,1 — not alternating)
