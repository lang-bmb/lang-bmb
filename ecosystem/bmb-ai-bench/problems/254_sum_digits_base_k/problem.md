# 254. Sum of Digits in Base K

Given an integer n (in base 10) and a base k, return the sum of the digits of n after converting n from base 10 to base k. (external, LeetCode #1837)

Input: first line is n, second line is k
Output: sum of digits in base k

Example:
Input:
34
6
Output:
9

Example:
Input:
10
10
Output:
1
