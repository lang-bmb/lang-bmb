# 303. Find Lucky Integer in an Array

Given an array of integers, a lucky integer is one whose value equals its frequency in the array. Return the largest lucky integer, or -1 if there is none. (external, LeetCode #1394)

Input: first line is n, then n integers (1..500, one per line)
Output: the largest lucky integer, or -1

Example:
Input:
4
1
2
2
3
Output:
2
