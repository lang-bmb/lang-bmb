# 251. Sign of the Product of an Array

There is a function signFunc(x) that returns 1 if x is positive, -1 if x is negative, and 0 if x is equal to 0. Given an integer array nums, return signFunc(product of all values in the array). (external, LeetCode #1822)

Input: first line is n, then n integers (one per line)
Output: 1, -1, or 0

Example:
Input:
3
-1
-2
-3
Output:
-1

Example:
Input:
2
1
5
Output:
1
