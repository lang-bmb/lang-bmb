# 206. Subtract the Product and Sum of Digits of an Integer

Given an integer n, return the difference between the product of its digits and the sum of its digits. (external, LeetCode #1281)

Input: a single integer n (1 <= n <= 10^5)
Output: product_of_digits - sum_of_digits

Example:
Input: 234
Output: 15

Input: 4421
Output: 21
