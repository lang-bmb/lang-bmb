# 192. Valid Palindrome II

Given a string s, return 1 if the s can be palindrome after deleting at most one character, 0 otherwise. (external, LeetCode #680)

Input: one string
Output: 1 if valid (at most one deletion), 0 otherwise

Example:
Input: aba
Output: 1

Input: abca
Output: 1
(remove 'c': "aba")

Input: abc
Output: 0
