# 203. N-th Tribonacci Number

The Tribonacci sequence T is defined as: T(0) = 0, T(1) = 1, T(2) = 1, and for n >= 3: T(n) = T(n-1) + T(n-2) + T(n-3). Given n, return the value of T(n). (external, LeetCode #1137)

Input: a single integer n (0 <= n <= 37)
Output: T(n)

Example:
Input: 4
Output: 4

Input: 25
Output: 1389537
