# Digit Sum

## Description

Compute the sum of digits of a non-negative integer.

**Input** (stdin):
- Single non-negative integer: `n` (0 <= n <= 10^18)

**Output** (stdout):
- Print the sum of digits of n

## Example

Input:
```
12345
```

Output:
```
15
```

## Constraints

- 0 <= n <= 10^18
- n fits in a 64-bit signed integer

## Category

System (number processing)
