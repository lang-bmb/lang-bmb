# 278. Find Target Indices After Sorting Array

Sort the array nums. Return a list of indices (sorted) where nums[index] == target. (external, LeetCode #2089)

Input: first line is n, second line is target, then n integers (one per line)
Output: indices where sorted nums equals target, space-separated

Example:
Input:
5
2
1
2
5
2
3
Output:
1 2

Example:
Input:
5
3
1
2
3
4
5
Output:
2
