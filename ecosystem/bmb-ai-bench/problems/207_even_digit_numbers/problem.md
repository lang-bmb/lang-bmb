# 207. Find Numbers with Even Number of Digits

Given an array nums of integers, return how many of them contain an even number of digits. (external, LeetCode #1295)

Input: first line n, then n integers (one per line)
Output: count of numbers with even number of digits

Example:
Input:
5
12
345
2
6
7896
Output: 2

Input:
5
555
901
482
1771
5
Output: 1
