# 320. Count Pairs With Given Sum

Given an array and a target, return the number of index pairs (i, j) with i < j such that nums[i] + nums[j] == target. (external-style)

Input: line 1 is n, line 2 is target, then n integers (one per line)
Output: the number of qualifying pairs

Example:
Input:
4
6
1
5
2
4
Output:
2
