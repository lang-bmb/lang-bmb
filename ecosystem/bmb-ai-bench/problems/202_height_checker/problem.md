# 202. Height Checker

Students are asked to stand in non-decreasing order of heights for an annual photo. Return the minimum number of students that must move in order for all students to be standing in non-decreasing order of height. (external, LeetCode #1051)

Input: first line n, then n integers (one per line)
Output: number of students out of position

Example:
Input:
6
1
1
4
2
1
3
Output: 3

Input:
3
5
1
2
Output: 2

Input:
3
1
2
3
Output: 0
