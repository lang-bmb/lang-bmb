# 275. Maximum Number of Words Found in Sentences

A sentence is a list of words separated by a single space. Given a string array sentences where each element is a sentence, return the maximum number of words that appear in a single sentence. (external, LeetCode #2114)

Input: first line is n (number of sentences), then n sentences (one per line)
Output: maximum word count

Example:
Input:
5
please wait
good luck alice and bob
look you can see it now
there is an apple on the tree
please
Output:
7

Example:
Input:
1
a
Output:
1
