# 215. Maximum Product of Two Elements in an Array

Given the array nums, choose two different indices i and j such that (nums[i]-1)*(nums[j]-1) is maximized. Return the maximum value of (nums[i]-1)*(nums[j]-1). (external, LeetCode #1464)

Input: first line n, then n integers (one per line)
Output: maximum product

Example:
Input:
4
3
4
5
2
Output:
12

Input:
3
1
5
4
Output:
12
