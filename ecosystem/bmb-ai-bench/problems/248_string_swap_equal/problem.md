# 248. Check if One String Swap Can Make Strings Equal

You are given two strings s1 and s2 of equal length. A string swap is an operation where you choose any two indices in a string and swap the characters at these indices. Return true if it is possible to make both strings equal by performing at most one string swap on exactly one of the strings. Otherwise, return false. (external, LeetCode #1790)

Input: two strings on separate lines
Output: "true" or "false"

Example:
Input:
bank
kanb
Output:
true

Example:
Input:
attack
defend
Output:
false
