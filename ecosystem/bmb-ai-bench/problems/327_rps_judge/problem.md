# 327. Rock Paper Scissors Judge

Two players each throw a move: 0 = Rock, 1 = Paper, 2 = Scissors. Return 0 for a tie, 1 if the first player wins, or 2 if the second player wins. (external-style, enum exercise)

Input: two integers (one per line): first player's move, then second player's move
Output: 0, 1, or 2

Example:
Input:
0
2
Output:
1
