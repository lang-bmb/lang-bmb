# 304. Count Elements With Strictly Smaller and Greater Elements

Given an integer array nums, return the number of elements that have both a strictly smaller and a strictly greater element appearing in nums. (external, LeetCode #2148)

Input: first line is n, then n integers (one per line)
Output: the count of qualifying elements

Example:
Input:
5
11
7
2
15
9
Output:
2
