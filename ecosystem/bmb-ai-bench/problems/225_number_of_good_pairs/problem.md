# 225. Number of Good Pairs

Given an array nums, a pair (i, j) is called good if nums[i] == nums[j] and i < j. Return the number of good pairs. (external, LeetCode #1512)

Input: first line n, then n integers (one per line)
Output: number of good pairs

Example:
Input:
4
1
2
3
1
Output:
1

Input:
5
1
1
1
1
1
Output:
10
