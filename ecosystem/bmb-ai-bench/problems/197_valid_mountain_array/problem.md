# 197. Valid Mountain Array

Given an array of integers, return 1 if it is a valid mountain array, 0 otherwise.
A valid mountain array satisfies: length >= 3, strictly increases to a peak, then strictly decreases. (external, LeetCode #941)

Input: first line n, then n integers (one per line)
Output: 1 if valid mountain, 0 otherwise

Example:
Input:
2
3
1
Output: 0

Input:
4
0
3
2
1
Output: 1

Input:
4
3
5
5
1
Output: 0
