# 177. Ransom Note

Given two strings ransomNote and magazine, return 1 if ransomNote can be constructed by using the letters from magazine, 0 otherwise. Each letter in magazine can only be used once. (external, LeetCode #383)

Input:
ransomNote
magazine

Output: 1 if possible, 0 otherwise

Example:
Input:
a
b
Output: 0

Input:
aa
ab
Output: 0

Input:
aa
aab
Output: 1
