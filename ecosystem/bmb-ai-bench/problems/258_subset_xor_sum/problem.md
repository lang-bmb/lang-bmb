# 258. Sum of All Subset XOR Totals

The XOR total of an array is the XOR of all its elements, or 0 if empty. Given an integer array nums, return the sum of all XOR totals for every subset of nums. (external, LeetCode #1863)

Note: Subsets with the same elements should be counted multiple times.

Input: first line is n, then n integers (one per line)
Output: sum of XOR totals of all subsets

Example:
Input:
3
1
3
5
Output:
28

Example:
Input:
1
7
Output:
7

Example:
Input:
6
3
4
5
6
7
8
Output:
480
