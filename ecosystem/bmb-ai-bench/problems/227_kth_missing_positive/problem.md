# 227. Kth Missing Positive Number

Given an array arr of positive integers sorted in strictly increasing order, and an integer k, return the kth positive integer that is missing from this array. (external, LeetCode #1539)

Input: first line n, then n sorted positive integers, then k
Output: the kth missing positive integer

Example:
Input:
4
2
3
4
7
2
Output:
5

Explanation: Missing positives are 1, 5, 6, 8, 9 ... so the 2nd missing is 5.
