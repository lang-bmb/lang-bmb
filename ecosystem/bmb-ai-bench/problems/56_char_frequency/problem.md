# Char Frequency

Output the frequency of each distinct value in sorted order.

## Input
- First integer: n
- Next n integers: the array

## Output
- First line: number of distinct values
- Then for each distinct value ascending: `value count` (one per line)

## Example
`5 1 2 1 3 2` -> 3 distinct; 1 appears 2x, 2 appears 2x, 3 appears 1x
Output:
```
3
1 2
2 2
3 1
```
