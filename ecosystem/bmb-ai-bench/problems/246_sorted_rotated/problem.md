# 246. Check if Array Is Sorted and Rotated

Given an array nums, return true if the array was originally sorted in non-decreasing order, then rotated some number of positions (including zero). Otherwise, return false. (external, LeetCode #1752)

Input: first line is n, then n integers (one per line)
Output: "true" or "false"

Example:
Input:
5
3
4
5
1
2
Output:
true

Example:
Input:
5
2
1
3
4
5
Output:
false
