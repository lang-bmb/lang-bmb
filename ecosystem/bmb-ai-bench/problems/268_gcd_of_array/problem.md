# 268. Find Greatest Common Divisor of Array

Given an integer array nums, return the greatest common divisor of the smallest number and largest number in nums. (external, LeetCode #1979)

Input: first line is n, then n integers (one per line)
Output: GCD of min and max

Example:
Input:
6
2
5
6
9
10
3
Output:
2

Example:
Input:
1
7
Output:
7
