# 265. Delete Characters to Make Fancy String

A fancy string is one where no three consecutive characters are equal. Given a string s, delete the minimum possible characters so that it becomes a fancy string. Return the resulting string. (external, LeetCode #1957)

Input: one line, string s
Output: the resulting fancy string

Example:
Input:
leeetcode
Output:
leetcode

Example:
Input:
aaabaaaa
Output:
aabaa

Example:
Input:
aab
Output:
aab
