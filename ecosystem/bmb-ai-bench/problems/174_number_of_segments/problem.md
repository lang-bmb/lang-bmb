# 174. Number of Segments in a String

Given a string s, return the number of segments in the string. A segment is defined to be a contiguous sequence of non-space characters. (external, LeetCode #434)

Input: one string (may contain spaces)
Output: number of segments

Example:
Input: Hello, my name is John
Output: 5

Input: Hello
Output: 1

Input: (empty or all spaces)
Output: 0
