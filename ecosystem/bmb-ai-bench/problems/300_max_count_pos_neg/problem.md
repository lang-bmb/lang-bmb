# 300. Maximum Count of Positive Integer and Negative Integer

Given a sorted integer array nums, return the maximum between the count of positive integers and the count of negative integers. (external, LeetCode #2529)

Input: first line is n, then n integers (one per line)
Output: max(count of positive, count of negative)

Example:
Input:
5
-2
-1
-1
1
2
Output:
3
