# 343. Minimum Running Balance

Starting from a balance of 0, apply a sequence of integer deltas one by one. Return the minimum balance reached at any point (including the starting 0). (external-style, simulation)

Input: line 1 is n, then n integer deltas (one per line)
Output: the minimum running balance

Example:
Input:
3
-2
3
-5
Output:
-4
