# 291. Count Asterisks

Given a string s, count asterisks '*' that are not between a pair of vertical bars '|'. (external, LeetCode #2315)

Input: one line, string s
Output: count of asterisks outside bars

Example:
Input:
l|*e*et|c*o*de|
Output:
2

Example:
Input:
iamprogrammer
Output:
0

Example:
Input:
yo|uar|e**|b|e***aut|iful
Output:
5
