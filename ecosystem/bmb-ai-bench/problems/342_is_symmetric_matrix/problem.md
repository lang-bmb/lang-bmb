# 342. Symmetric Matrix Check

Given an n x n matrix, return 1 if it is symmetric (a[i][j] == a[j][i] for all i, j), otherwise 0. (external-style, 2D exercise)

Input: line 1 is n, then n*n integers in row-major order (one per line)
Output: 1 if symmetric, else 0

Example:
Input:
2
1
2
2
1
Output:
1
