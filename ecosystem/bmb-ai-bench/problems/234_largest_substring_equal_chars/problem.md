# 234. Largest Substring Between Two Equal Characters

Given a string s, return the length of the longest substring between two equal characters, excluding the two characters. If there is no such substring return -1.

A substring is a contiguous sequence of characters within a string. (external, LeetCode #1624)

Input: a single string
Output: length of longest substring between two equal characters, or -1

Example:
Input:
aa
Output:
0

Example:
Input:
abca
Output:
2
