# 264. Check if All Characters Have Equal Number of Occurrences

Given a string s, return 1 if all the characters in s have the same number of occurrences, or 0 otherwise. (external, LeetCode #1941)

Input: one line, lowercase string s
Output: 1 if all chars have equal occurrences, else 0

Example:
Input:
abacbc
Output:
1

Example:
Input:
aaabb
Output:
0
