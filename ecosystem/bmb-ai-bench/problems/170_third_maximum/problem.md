# 170. Third Maximum Number

Given an array of integers, return the third distinct maximum. If it doesn't exist, return the maximum. (external, LeetCode #414)

Input:
n (array length)
n integers each on a separate line

Output: third distinct maximum or maximum

Example:
Input:
3
3
2
1
Output: 1

Input:
3
1
2
2
Output: 2
(only 2 distinct: 1,2 → return max=2)

Input:
5
2
2
3
1
1
Output: 1
