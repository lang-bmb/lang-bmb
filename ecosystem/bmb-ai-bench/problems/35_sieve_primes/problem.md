# Sieve Primes

## Description

Count the number of prime numbers up to n (inclusive) using the Sieve of Eratosthenes.

**Input** (stdin):
- A single integer `n` (0 <= n <= 10000000)

**Output** (stdout):
- Print the count of primes in [2, n]

## Example

Input:
```
10
```

Output:
```
4
```

(Primes up to 10: 2, 3, 5, 7 → count = 4)

## Constraints

- 0 <= n <= 10000000

## Category

Algorithm (prime sieve)
