# Longest Increasing

Find the length of the Longest Strictly Increasing Subsequence (LIS).

## Input
- First integer: n
- Next n integers: the sequence

## Output
Length of the longest strictly increasing subsequence.

## Examples
`6 10 9 2 5 3 7` -> LIS=[2,5,7] or [2,3,7], length=3
`5 1 2 3 4 5` -> 5
