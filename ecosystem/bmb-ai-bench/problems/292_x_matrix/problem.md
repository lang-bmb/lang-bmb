# 292. Check if Matrix Is X-Matrix

A square matrix is an X-Matrix if: all elements on main diagonals are nonzero, and all other elements are 0. Return 1 if the matrix is an X-Matrix, else 0. (external, LeetCode #2319)

Input: first line is n, then n*n integers (one per line, row by row)
Output: 1 if X-Matrix, else 0

Example:
Input:
3
5
0
3
0
4
0
1
0
7
Output:
1

Example:
Input:
3
2
0
0
0
1
0
0
0
3
Output:
0
