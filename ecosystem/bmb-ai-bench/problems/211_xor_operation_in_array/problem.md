# 211. XOR Operation in an Array

Given two integers n and start. Define an array nums where nums[i] = start + 2*i (0-indexed) and n is the length of the array. Return the bitwise XOR of all elements of nums. (external, LeetCode #1486)

Input: two integers n and start (one per line)
Output: XOR of all elements

Example:
Input:
5
0
Output: 8

Input:
4
3
Output: 8

Input:
1
7
Output: 7
