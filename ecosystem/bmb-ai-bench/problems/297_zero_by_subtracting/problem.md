# 297. Make Array Zero by Subtracting Equal Amounts

Given a non-negative integer array nums. In one operation, choose a positive integer x and subtract x from every positive element. Return the minimum number of operations to make all elements 0. (external, LeetCode #2357)

Input: first line is n, then n integers (one per line)
Output: minimum operations (= number of distinct non-zero values)

Example:
Input:
6
1
5
0
3
5
0
Output:
3

Example:
Input:
3
0
0
0
Output:
0
