# 328. Closest Point to Origin

Given n points on a 2D plane, return the smallest squared Euclidean distance (x*x + y*y) from the origin among all points. (external-style, struct exercise)

Input: line 1 is n, then for each point two integers x and y (one per line)
Output: the minimum squared distance

Example:
Input:
2
1
3
-2
2
Output:
8
