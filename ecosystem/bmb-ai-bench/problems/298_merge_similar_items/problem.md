# 298. Number of Arithmetic Triplets

Given a sorted integer array nums and a positive integer diff, return the number of unique arithmetic triplets (i,j,k) where i<j<k and nums[j]-nums[i]==nums[k]-nums[j]==diff. (external, LeetCode #2367)

Input: first line is n, then n integers (one per line), then diff
Output: count of arithmetic triplets

Example:
Input:
6
0
1
4
6
7
10
3
Output:
2

Example:
Input:
4
4
5
6
7
2
Output:
0
