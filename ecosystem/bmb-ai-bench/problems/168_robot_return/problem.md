# 168. Robot Return to Origin

A robot starts at (0,0). Moves: U(+y), D(-y), L(-x), R(+x). Given a sequence, check if it returns to origin. (external, LeetCode #657)

Input: string of moves (U/D/L/R characters)
Output: 1 if returns to origin, 0 otherwise

Example:
Input: UD
Output: 1

Input: LL
Output: 0
