# 305. Minimum Operations to Make the Array Increasing

Given an integer array nums, in one operation you can increment any element by 1. Return the minimum number of operations needed to make the array strictly increasing. (external, LeetCode #1827)

Input: first line is n, then n integers (one per line)
Output: minimum number of increment operations

Example:
Input:
3
1
1
1
Output:
3
