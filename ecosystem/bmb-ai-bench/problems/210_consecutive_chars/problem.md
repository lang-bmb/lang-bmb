# 210. Consecutive Characters

Given a string s, the power of the string is the maximum length of a non-empty substring that contains only one unique character. Return the power of the string. (external, LeetCode #1446)

Input: a single string s
Output: maximum run length of same character

Example:
Input: leetcode
Output: 2

Input: abbcccddddeeeeedcba
Output: 5

Input: triplepillooooow
Output: 5
