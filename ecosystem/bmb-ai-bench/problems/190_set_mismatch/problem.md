# 190. Set Mismatch

You have a set of integers [1, n]. One integer was replaced by a duplicate. Find the duplicate and the missing number. Output "dup miss" on one line. (external, LeetCode #645)

Input: first line n, then n integers (one per line)
Output: "dup miss" where dup is the duplicate and miss is the missing number

Example:
Input:
4
1
2
2
4
Output: 2 3

Input:
2
1
1
Output: 1 2
