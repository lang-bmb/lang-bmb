# 228. Three Consecutive Odds

Given an integer array arr, return 1 if there are three consecutive odd numbers in the array. Otherwise, return 0. (external, LeetCode #1550)

Input: first line n, then n integers
Output: 1 or 0

Example:
Input:
5
2
6
4
1
3
Output:
0

Example:
Input:
7
1
3
5
0
1
0
1
Output:
1
