# 274. Check if Numbers Are Ascending in a Sentence

Given a string s representing a sentence with tokens separated by spaces, return 1 if all the numbers in s are strictly ascending from left to right, or 0 otherwise. (external, LeetCode #2042)

Input: one line, a sentence
Output: 1 if numbers are strictly ascending, else 0

Example:
Input:
1 box has 3 blue 4 red 6 green and 12 yellow marbles
Output:
1

Example:
Input:
hello world 5 x 5
Output:
0

Example:
Input:
sunset is at 7 51 pm overnight lows will be in the low 50 and 60 s
Output:
0
