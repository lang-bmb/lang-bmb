# 239. Count the Number of Consistent Strings

You are given a string allowed consisting of distinct characters and an array of strings words. A string is consistent if all characters in the string appear in the string allowed.

Return the number of consistent strings in the array words. (external, LeetCode #1684)

Input: first line is allowed string, second line is n, then n strings
Output: count of consistent strings

Example:
Input:
ab
3
ad
bd
aaab
Output:
2
