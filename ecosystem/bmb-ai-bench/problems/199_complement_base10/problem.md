# 199. Complement of Base 10 Integer

The complement of an integer is the integer obtained by flipping all the bits in its binary representation. Given a non-negative integer n, return its complement. (external, LeetCode #1009)

Input: a single integer n (0 <= n <= 10^9)
Output: its complement

Example:
Input: 5
Output: 2

Input: 7
Output: 0

Input: 10
Output: 5
