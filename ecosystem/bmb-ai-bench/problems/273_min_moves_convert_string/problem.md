# 273. Minimum Moves to Convert String

You are given a string s of length n consisting of 'X' and 'O'. In one operation, you can select three consecutive characters and convert them to 'O'. Return the minimum number of operations to convert all 'X' to 'O'. (external, LeetCode #2027)

Input: one line, string of X and O
Output: minimum operations

Example:
Input:
XXX
Output:
1

Example:
Input:
XXOX
Output:
2

Example:
Input:
OOOO
Output:
0
