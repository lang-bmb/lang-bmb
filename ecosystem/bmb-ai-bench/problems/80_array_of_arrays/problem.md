# Array of Arrays

Given a matrix with `rows` rows and `cols` columns, compute column sums.

## Input
- First integer: rows
- Second integer: cols
- Next rows*cols integers: matrix row-major

## Output
Column sums, space-separated on one line.

## Example
`2 3 1 2 3 4 5 6` -> [[1,2,3],[4,5,6]] -> col sums: 5 7 9
