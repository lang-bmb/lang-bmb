# 259. Longer Contiguous Segments of Ones than Zeros

Given a binary string s, return 1 if the longest contiguous segment of 1s is strictly longer than the longest contiguous segment of 0s in s, or return 0 otherwise. (external, LeetCode #1869)

Input: one line, a binary string s
Output: 1 if longest run of 1s > longest run of 0s, else 0

Example:
Input:
1101
Output:
1

Example:
Input:
111000
Output:
0

Example:
Input:
110100010
Output:
0
