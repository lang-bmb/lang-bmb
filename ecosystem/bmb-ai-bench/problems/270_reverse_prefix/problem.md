# 270. Reverse Prefix of Word

Given a 0-indexed string word and a character ch, find the first occurrence of ch in word and reverse the segment from index 0 to that index (inclusive). If ch does not exist, do nothing and return the original word. (external, LeetCode #2000)

Input: first line is word, second line is the character ch
Output: resulting string

Example:
Input:
abcdefd
d
Output:
dcbaefd

Example:
Input:
xyxzxe
z
Output:
zxyxxe

Example:
Input:
abcd
z
Output:
abcd
