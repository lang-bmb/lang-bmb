# 250. Truncate Sentence

A sentence is a list of words that are separated by a single space with no leading or trailing spaces. You are given a sentence s and an integer k. You want to truncate sentence s such that it contains only the first k words. Return s after truncating it. (external, LeetCode #1816)

Input: first line is the sentence, second line is k
Output: truncated sentence

Example:
Input:
Hello how are you Contestant
4
Output:
Hello how are you

Example:
Input:
chopper is not a tanuki
5
Output:
chopper is not a tanuki
