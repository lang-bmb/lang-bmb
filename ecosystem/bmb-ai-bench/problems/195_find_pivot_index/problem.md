# 195. Find Pivot Index

Given an array of integers nums, calculate the pivot index where the sum of all numbers strictly to the left equals the sum to the right. Return the leftmost pivot index, or -1. (external, LeetCode #724)

Input: first line n, then n integers (one per line)
Output: leftmost pivot index or -1

Example:
Input:
7
1
7
3
6
5
6
0
Output: 3

Input:
3
1
2
3
Output: -1
