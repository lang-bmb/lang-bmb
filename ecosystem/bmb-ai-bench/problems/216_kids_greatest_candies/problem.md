# 216. Kids With the Greatest Number of Candies

Given an array candies and an integer extraCandies, for each kid, determine if they can have the greatest (or equal) number of candies among all kids after receiving the extraCandies. (external, LeetCode #1431)

Input: first line n (number of kids), then n integers (candies per kid, one per line), then extraCandies
Output: for each kid, 1 if they can have the greatest, 0 otherwise (one per line)

Example:
Input:
5
2
3
5
1
3
3
Output:
1
1
1
0
1

Input:
3
4
2
1
1
Output:
1
0
0
