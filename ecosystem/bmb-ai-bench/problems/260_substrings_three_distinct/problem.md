# 260. Substrings of Size Three with Distinct Characters

Given a string s, return the number of good substrings of length 3. A substring is good if all three characters are distinct. (external, LeetCode #1876)

Input: one line, string s
Output: count of 3-length substrings with all distinct characters

Example:
Input:
xyzzaz
Output:
1

Example:
Input:
aababcabc
Output:
4
