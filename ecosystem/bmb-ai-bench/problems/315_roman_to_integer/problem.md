# 315. Roman to Integer

Given a Roman numeral string, convert it to an integer. Symbols: I=1, V=5, X=10, L=50, C=100, D=500, M=1000. A smaller value before a larger one is subtracted. (external, LeetCode #13)

Input: a single Roman numeral string
Output: the integer value

Example:
Input:
MCMXCIV
Output:
1994
