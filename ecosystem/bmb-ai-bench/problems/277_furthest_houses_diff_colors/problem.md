# 277. Two Furthest Houses With Different Colors

There are n houses evenly placed in a row. Each has a color. Find the maximum distance between two houses with different colors. (external, LeetCode #2078)

Input: first line is n, then n color values (one per line)
Output: maximum distance

Example:
Input:
7
1
1
1
6
1
1
1
Output:
3

Example:
Input:
5
1
8
3
8
3
Output:
4
