# 262. Build Array from Permutation

Given a zero-based permutation nums (0-indexed), build an array ans of the same length where ans[i] = nums[nums[i]] for each 0 <= i < nums.length. Return the array. (external, LeetCode #1920)

Input: first line is n, then n integers (one per line)
Output: n integers space-separated on one line

Example:
Input:
4
0
2
1
5
3
Output:
0 1 2 5

Note: For this problem, the input gives a 0-indexed permutation. The output shows ans[i] = nums[nums[i]] space-separated.
