# 287. Largest 3-Same-Digit Number in String

Given a string num representing a large integer, return the largest good integer which is a substring of num. A good integer is a string of three equal digits. Return "" if no good integer exists. (external, LeetCode #2264)

Input: one line, string of digits
Output: the largest good integer substring, or empty line if none

Input:
6777133339
Output:
777
