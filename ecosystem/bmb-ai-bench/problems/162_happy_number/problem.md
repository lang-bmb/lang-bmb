# 162. Happy Number

A number is happy if repeatedly replacing it with the sum of squares of its digits eventually reaches 1. If it never reaches 1 (loops), it is not happy. (external, LeetCode #202)

Input: integer n
Output: 1 if happy, 0 otherwise

Example:
Input: 19
Output: 1
(19 → 1²+9²=82 → 8²+2²=68 → 6²+8²=100 → 1)

Input: 2
Output: 0
