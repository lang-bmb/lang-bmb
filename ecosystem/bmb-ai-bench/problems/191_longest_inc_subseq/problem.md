# 191. Longest Continuous Increasing Subsequence

Given an unsorted array of integers, find the length of longest continuous increasing subsequence (subarray). (external, LeetCode #674)

Input: first line n, then n integers (one per line)
Output: length of longest continuous increasing subarray

Example:
Input:
6
1
3
5
4
7
1
Output: 3

Explanation: [1,3,5] has length 3.

Input:
5
2
2
2
2
2
Output: 1
