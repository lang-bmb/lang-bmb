# 340. Caesar Cipher

Given a string and a non-negative shift k, return the string with each alphabetic character shifted forward by k positions in the alphabet (wrapping around), preserving case. Non-alphabetic characters are unchanged. (external-style, string-build exercise)

Input: line 1 is the text, line 2 is the shift k (k >= 0)
Output: the encrypted string

Example:
Input:
abc
1
Output:
bcd
