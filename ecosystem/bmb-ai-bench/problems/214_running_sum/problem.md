# 214. Running Sum of 1d Array

Given an array nums, return the running sum of nums. The running sum of element i is sum(nums[0] + nums[1] + ... + nums[i]). (external, LeetCode #1480)

Input: first line n, then n integers (one per line)
Output: running sum for each element (one per line)

Example:
Input:
4
1
2
3
4
Output:
1
3
6
10

Input:
5
1
1
1
1
1
Output:
1
2
3
4
5
