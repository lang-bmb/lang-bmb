# 344. Josephus Survivor

There are n people standing in a circle, numbered 0 to n-1. Counting starts at person 0; every k-th person is eliminated until one remains. Return the 0-based index of the survivor. (external, Josephus problem)

Input: two integers (one per line): n (>= 1), then k (>= 1)
Output: the 0-based index of the survivor

Example:
Input:
5
2
Output:
2
