# Safe Matrix Mul

Multiply two n x n matrices.

## Input
- First integer: n
- Next 2*n*n integers: matrix A (row-major) then matrix B (row-major)

## Output
C = A*B, each row space-separated on its own line.

## Example
`2 1 2 3 4 5 6 7 8` -> A=[[1,2],[3,4]], B=[[5,6],[7,8]] -> C=[[19,22],[43,50]]
