# 173. Is Subsequence

Given two strings s and t, return 1 if s is a subsequence of t, or 0 otherwise. A subsequence is a string formed by deleting some (or no) characters without changing the relative order. (external, LeetCode #392)

Input:
s (the candidate subsequence)
t (the target string)

Output: 1 if s is a subsequence of t, 0 otherwise

Example:
Input:
abc
ahbgdc
Output: 1

Input:
axc
ahbgdc
Output: 0

Input:
(empty line)
ahbgdc
Output: 1
(empty string is subsequence of anything)
