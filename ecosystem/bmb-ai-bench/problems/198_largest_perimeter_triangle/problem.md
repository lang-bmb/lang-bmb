# 198. Largest Perimeter Triangle

Given an integer array, return the largest perimeter of a triangle with a non-zero area formed from three of these values. If no such triangle exists, return 0. (external, LeetCode #976)

Input: first line n, then n integers (one per line)
Output: largest valid perimeter, or 0 if none exists

Example:
Input:
3
2
1
2
Output: 5

Input:
3
1
2
1
Output: 0

Input:
4
3
2
3
4
Output: 10
