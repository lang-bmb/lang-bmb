# 280. Minimum Sum of Four Digit Number After Splitting

Given a positive four-digit integer num, split its digits into two new integers new1 and new2 using all four digits. Return the minimum possible sum. Leading zeros are allowed. (external, LeetCode #2160)

Input: one line, a 4-digit integer
Output: minimum sum

Example:
Input:
2932
Output:
52

Example:
Input:
4009
Output:
13
