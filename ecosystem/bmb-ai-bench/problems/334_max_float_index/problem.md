# 334. Index of Maximum Float

Given n floating-point numbers, return the 0-based index of the maximum value. If several values are equal to the maximum, return the index of the first one. (external-style, f64 exercise)

Input: line 1 is n, then n floating-point numbers (one per line)
Output: the 0-based index of the maximum value

Example:
Input:
4
1.5
3.2
0.9
2.7
Output:
1
