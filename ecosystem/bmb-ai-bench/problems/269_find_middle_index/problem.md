# 269. Find the Middle Index in Array

Given a 0-indexed integer array nums, find the leftmost middleIndex such that the sum of elements to the left equals the sum of elements to the right. Return -1 if no such index exists. (external, LeetCode #1991)

Input: first line is n, then n integers (one per line)
Output: leftmost middle index or -1

Example:
Input:
6
2
3
-1
8
4
0
Output:
3

Example:
Input:
3
1
-1
4
Output:
2

Example:
Input:
3
2
5
3
Output:
-1
