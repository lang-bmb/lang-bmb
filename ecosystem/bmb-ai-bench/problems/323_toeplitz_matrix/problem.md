# 323. Toeplitz Matrix

A matrix is Toeplitz if every diagonal from top-left to bottom-right has the same elements. Return 1 if the matrix is Toeplitz, otherwise 0. (external, LeetCode #766)

Input: line 1 is m, line 2 is n, then m*n integers in row-major order (one per line)
Output: 1 if Toeplitz, else 0

Example:
Input:
3
4
1
2
3
4
5
1
2
3
9
5
1
2
Output:
1
