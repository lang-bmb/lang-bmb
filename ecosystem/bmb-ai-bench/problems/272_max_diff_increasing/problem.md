# 272. Maximum Difference Between Increasing Elements

Given a 0-indexed integer array nums of n elements, find the maximum difference between nums[i] and nums[j] where 0 <= i < j < n and nums[i] < nums[j]. Return -1 if no such pair exists. (external, LeetCode #2016)

Input: first line is n, then n integers (one per line)
Output: maximum difference or -1

Example:
Input:
6
7
1
5
4
8
2
Output:
7

Example:
Input:
7
9
4
3
2
1
0
-1
Output:
-1
