# 193. Baseball Game

You are keeping score for a baseball game with strange rules. Given a list of operations, calculate the total sum of all scores. Operations: integer score, C (remove last), D (double last), + (add last two). (external, LeetCode #682)

Input: first line n (number of ops), then n lines each with an operation
Output: total sum

Example:
Input:
7
5
2
C
D
+
C
+
Output: 30

Input:
5
5
-2
4
C
+
Output: 10
