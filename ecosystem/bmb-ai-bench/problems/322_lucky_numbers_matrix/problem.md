# 322. Lucky Numbers in a Matrix

Given an m x n matrix of distinct numbers, a lucky number is an element that is the minimum in its row and the maximum in its column. Return the lucky number, or -1 if none exists. (external, LeetCode #1380)

Input: line 1 is m, line 2 is n, then m*n integers in row-major order (one per line)
Output: the lucky number, or -1

Example:
Input:
3
3
3
7
8
9
11
13
15
16
17
Output:
15
