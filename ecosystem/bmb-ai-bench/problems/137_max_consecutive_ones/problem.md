# 137. Max Consecutive Ones

LeetCode #485

## Problem

Given a binary array (containing only 0s and 1s), return the maximum number of consecutive 1s.

## Input

- Line 1: n
- Next n lines: 0 or 1

## Output

Maximum consecutive ones

## Examples

```
Input:
5
1
1
0
1
1
Output: 2
```
