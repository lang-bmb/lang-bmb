# 245. Sum of Unique Elements

You are given an integer array nums. The unique elements of an array are the elements that appear exactly once in the array. Return the sum of all the unique elements of nums. (external, LeetCode #1748)

Input: first line is n, then n integers (one per line)
Output: sum of unique elements

Example:
Input:
5
1
2
3
2
1
Output:
3

Example:
Input:
5
1
2
3
4
5
Output:
15
