# 224. Destination City

You are given the array paths, where paths[i] = [cityAi, cityBi] means there exists a direct path from cityAi to cityBi. Return the destination city, that is, the city without any path outgoing from it. (external, LeetCode #1436)

Input: first line n (number of paths), then for each path: cityA on one line, cityB on one line
Output: the destination city name

Example:
Input:
3
London
New York City
New York City
Lima
Lima
Sao Paulo
Output:
Sao Paulo

Input:
2
D
B
B
Z
Output:
Z
