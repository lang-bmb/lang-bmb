# 249. Maximum Ascending Subarray Sum

Given an array of positive integers nums, return the maximum possible sum of an ascending subarray in nums. A subarray is defined as a contiguous sequence of numbers in an array. A subarray [nums_l, nums_l+1, ..., nums_r-1, nums_r] is ascending if for all i where l <= i < r, nums_i < nums_i+1. Note that a subarray of size 1 is ascending. (external, LeetCode #1800)

Input: first line is n, then n integers (one per line)
Output: maximum ascending subarray sum

Example:
Input:
5
10
20
30
5
10
Output:
60

Example:
Input:
3
12
17
15
Output:
29
