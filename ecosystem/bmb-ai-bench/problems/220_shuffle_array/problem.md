# 220. Shuffle the Array

Given an array nums consisting of 2n elements in the form [x1,x2,...,xn,y1,y2,...,yn], return the array in the form [x1,y1,x2,y2,...,xn,yn]. (external, LeetCode #1470)

Input: first line n, then 2n integers (one per line)
Output: shuffled array elements (one per line)

Example:
Input:
3
2
5
1
3
4
4
Output:
2
3
5
4
1
4

Input:
2
1
2
3
4
Output:
1
3
2
4
