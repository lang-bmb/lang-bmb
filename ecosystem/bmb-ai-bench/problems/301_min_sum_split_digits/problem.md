# 301. Minimum Sum of Four Digit Number After Splitting Digits

Given a four-digit integer num, split its digits into two new integers (each using two digits, leading zeros allowed) so the sum of the two integers is minimized. Return that minimum sum. (external, LeetCode #2160)

Input: a single four-digit integer (1000..9999)
Output: the minimum possible sum

Example:
Input:
2932
Output:
52
