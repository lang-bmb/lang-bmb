# 267. Minimum Time to Type Word Using Special Typewriter

A special typewriter has a circular wheel with lowercase English letters. The wheel starts at 'a'. You can turn the wheel in either direction. Each turn by one step costs 1 second, and typing the letter costs 1 second. Given a string word, return the minimum number of seconds to type word. (external, LeetCode #1974)

Input: one line, string word
Output: minimum seconds

Example:
Input:
abc
Output:
5

Example:
Input:
bza
Output:
7

Example:
Input:
zjpc
Output:
34
