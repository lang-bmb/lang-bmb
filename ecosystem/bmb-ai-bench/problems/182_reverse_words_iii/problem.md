# 182. Reverse Words in a String III

Given a string s, reverse the order of characters in each word within a sentence while still preserving whitespace and initial word order. (external, LeetCode #557)

Input: one string with words separated by single spaces
Output: string with each word reversed

Example:
Input: Let's take LeetCode contest
Output: s'teL ekat edoCteeL tsetnoC

Input: God Ding
Output: doG gniD
