# 208. Number of Steps to Reduce a Number to Zero

Given a non-negative integer num, return the number of steps to reduce it to zero. If the current number is even, divide it by 2; otherwise, subtract 1 from it. (external, LeetCode #1342)

Input: a single integer num (0 <= num <= 10^6)
Output: number of steps

Example:
Input: 14
Output: 6

Input: 8
Output: 4

Input: 123
Output: 12
